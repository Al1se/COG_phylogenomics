Default destination for analysis runs produced by `sh/run_full_analysis.sh`.

Suggested naming pattern:
- `results/COG0085_*`
- `results/COG0086_*`
- `results/COG0129_*`
- `results/COG1594_*`
- `results/COG2093_*`

Typical contents of a run directory:
- extracted COG FASTA files,
- structure token lists and reports,
- downloaded structure sequences,
- combined and core alignments,
- `tree.nwk`,
- `reordered.fasta`,
- structure-contact and site-coordinate reports.
